1. Struktura e Sistemit

1.1 Pamje e Përgjithshme

Website-i InfoMat është ndërtuar mbi një arkitekturë moderne tre-shtresore:

┌─────────────────────────────────────────────────────┐
│                    SISTEMI InfoMat                  │
├─────────────────────────────────────────────────────┤
  FRONTEND   →  Next.js 16 (React) · TypeScript
             →  infomat.app
├─────────────────────────────────────────────────────┤
  CMS ADMIN  →  Sanity Studio
             →  infomat.app/studio
├─────────────────────────────────────────────────────┤
  HOSTING    →  Vercel (deploy automatik)
  DOMAIN     →  Cloudflare (DNS)
  KOD        →  GitHub (github.com/aplikimek/infomat-web)
└─────────────────────────────────────────────────────┘
Si funksionon lidhja Kur ndryshoni diçka në Sanity Studio, të dhënat ruhen në cloud-in e Sanity dhe faqja i merr automatikisht pa pasur nevojë për ri-deployment.

2. Paneli i Administrimit (Sanity Studio)

2.1 Hyrja në Panel

1
Hapni shfletuesin dhe shkoni te: https://infomat.app/studio
2
Klikoni "Log in with" → zgjidhni llogarinë tuaj Google ose GitHub (të njëjtën me të cilën jeni regjistruar në sanity.io)
3
Klikoni "Structure" në shiritin e sipërm për të parë të gjitha seksionet

Paneli juaj i administrimit ka këtë strukturë:

⚙️

2.2 Ndryshimi i Cilësimeve dhe Menusë

1
Klikoni ⚙️ Cilësimet & Menu
2
Ndryshoni fushat sipas nevojës:
  • Emri i Faqes – shfaqet si titull i browser-it
  • Slogani – teksti i shkurtër nën logo në footer
  • Email Kontakti – adresa email që shfaqet në footer dhe kontakt
  • GitHub / LinkedIn URL – linqet e rrjeteve sociale
  • Menu (Navigimi) – klikoni "Add item" për të shtuar lidhje të reja
3
Klikoni "Publish" (butoni i gjelbër lart djathtas) për të ruajtur ndryshimet
⚠️ Kujdes Ndryshimet ruhen si Draft derisa të klikoni Publish. Nëse mbyllni faqen pa Publish, ndryshimet humbasin.

2.3 Editimi i Hero Seksionit

Hero seksioni është pjesa e parë që sheh vizitori kur hap faqen kryesore.

1
Klikoni 🏠 Hero (Faqja Kryesore)
2
Ndryshoni fushat:
  • Badge teksti – teksti i vogël mbi titull (p.sh. "MATEMATIKË E APLIKUAR")
  • Titulli Rreshti 1 – fjala/frazë e parë (p.sh. "Ku Matematika")
  • Titulli Rreshti 2 me ngjyrë – fraza me ngjyrë blu (p.sh. "Takon Aplikimin")
  • Nëntitulli – teksti shpjegues nën titull
  • Butoni kryesor / dytësor – teksti i butonëve
  • Imazhi Hero – imazhi që shfaqet djathtas (klikoni "Upload" ose "Select")
3
Plotësoni të njëjtat fusha edhe për anglisht (fushat me mbiemrin "EN")
4
Klikoni Publish

2.4 Menaxhimi i Projekteve

Shtimi i Projektit të Ri

1
Klikoni 📁 Projektet → butoni "+" lart djathtas
2
Plotësoni formularin:
  • Titulli (EN/SQ) – emri i projektit
  • Kategoria – zgjidhni: GIS / Operational Research / Statistics / Visualization
  • Përshkrimi (EN/SQ) – teksti shpjegues
  • Etiketat – teknologjitë e përdorura (Python, QGIS, etj.) – shtypni Enter pas çdo etikete
  • Imazhi – screenshot i projektit
  • Linku Live – URL e projektit live (opsionale)
  • Kodi Burimor – URL e GitHub (opsionale)
  • Renditja – numri i renditjes (0 = i pari)
3
Klikoni Publish

Editimi i Projektit Ekzistues

  • Klikoni projektin nga lista → bëni ndryshimet → klikoni Publish

Fshirja e Projektit

  • Hapni projektin → klikoni "..." (tre pikat) lart djathtas → "Delete"
⛔ Fshirja është e pakthyeshme! Pasi fshini një projekt, nuk mund ta riktheni. Sigurohuni para se të klikoni Delete.

2.5 Shtimi i Kurseve

1
Klikoni 🎓 Kurset → butoni "+"
2
Plotësoni:
  • Titulli (SQ/EN) – emri i kursit
  • Përshkrimi (SQ/EN) – çfarë mbulon kursi
  • Niveli – Fillues / Mesatar / Avancuar
  • Kohëzgjatja – p.sh. "8 javë" / "8 weeks"
  • Linku i Kursit – URL ku mund ta aksesojnë studentët (opsionale)
  • Renditja – numri i renditjes në faqe
3
Klikoni Publish
ℹ️ Renditja e kurseve Kurset renditen sipas fushës Renditja. Vendosni 1 për kursin e parë, 2 për të dytin, etj. Nëse dy kurse kanë të njëjtin numër, renditen sipas datës së krijimit.

Plotësoni fushat:

ℹ️ Email dhe Linqet Sociale (faqja Contact) Email-i, GitHub-u dhe LinkedIn-i që shfaqen në faqen Contact menaxhohen nga ⚙️ Cilësimet e Faqes → fushat Email Kontakti, GitHub URL, LinkedIn URL.

2.11 Ngarkimi i Imazheve

Imazhet mund t'i ngarkoni në çdo fushë të tipit "Image" në Studio:

1
Klikoni fushën e imazhit
2
Zgjidhni një opsion:
  • Upload – ngarkoni nga kompjuteri juaj
  • Select – zgjidhni nga imazhet e ngarkohet më parë
3
Vendosni pikën e fokusit duke klikuar mbi imazhin (rrathë i vogël)
✅ Rekomandimet për imazhe
  • Formati: JPG ose PNG (WebP pranohet gjithashtu)
  • Madhësia: maksimumi 2MB për performancë optimale
  • Dimensionet: minimum 800×600px për hero, 400×300px për projekte
  • Sanity i optimizon dhe i kthen automatikisht në formatin WebP

3.1 Hyrja në Cloudflare

Shkoni te https://dash.cloudflare.com

3.2 DNS Records

DNS records aktuale të konfiguruara për infomat.app:

TypeNameContentProxyQëllimi
A@76.76.21.21DNS OnlyRoot domain → Vercel
CNAMEwwwcname.vercel-dns.comDNS Onlywww → Vercel
TXT_vercelvc-domain-verify=...DNS OnlyVerifikimi i Vercel
⚠️ Mos ndryshoni DNS records Nëse ndryshoni ose fshini këto records, website-i mund të dalë jashtë linje. Ndryshoni vetëm nëse Vercel ju kërkon specifike.

Shtimi i Email-it (opsionale)

Nëse dëshironi të merrni email në info@infomat.app, duhet të shtoni MX records. Rekomandueset shërbime falas:

  • Cloudflare Email Routing – ridrejton emailet te Gmail juaj (falas)
  • Google Workspace – email profesional (me pagesë)

Për Cloudflare Email Routing:

1
Menuja e majtë → Email → Email Routing
2
Aktivizoni dhe shtoni rregullin: info@infomat.app → sh.kuka@gmail.com
3
Cloudflare shton automatikisht MX records

4. Deployment (Vercel)

4.1 Deploy Automatik

Çdo herë që bëni git push në GitHub, Vercel ndez automatikisht një deploy të ri:

# Procesi automatik:
Git Push → GitHub → Vercel zbuloni ndryshimin
         → Build Next.js (~2 min)
         → Deploy në infomat.app

Mund ta ndiqni statusin e deploy-it te: https://vercel.com/dashboard

4.2 Ndryshimi i Kodit dhe Ri-Deployment

Kur bëni ndryshime në kod nga VSCode / Claude Code:

# 1. Ruani ndryshimet e skedarëve
# 2. Hapni terminal dhe ekzekutoni:

git add .
git commit -m "Përshkrimi i ndryshimit"
git push origin main

# Vercel fillon automatikisht deploy-in
ℹ️ Direktoria e Projektit Kodi burimor gjendet te: C:\Projekte\website\mywebsite\

4.3 Variablat e Mjedisit (Environment Variables)

Nëse keni nevojë të shtoni çelësa API ose konfigurime sekrete:

1
Shkoni te vercel.com/dashboard → projekti infomat-web
2
Settings → Environment Variables
3
Shtoni variablin: KEY = VALUE → zgjidhni Production → Save
VariableVleraQëllimi
NEXT_PUBLIC_SANITY_PROJECT_IDn6umd8u9ID e projektit Sanity
NEXT_PUBLIC_SANITY_DATASETproductionDataset-i Sanity
✅ Rifreskim pas ndryshimit të variables Pas shtimit të variablave të reja, bëni një Redeploy manual te Vercel: Deployments → tre pikat → Redeploy.
